# かんたん天気
自分用の天気を表示するだけのページ。

## 実装した内容

* 位置情報：ブラウザーのGPS → OpenStreetMap逆ジオコーディングで地名を自動取得。拒否した場合や失敗時は保存済みCookieか東京にフォールバック。
* 都市検索：モーダルから都市名を入力して切り替え可能。選択位置はCookieに30日保存。
* 天気データ：Open-Meteoを使用。
* 表示内容：現在の気温、体感温度、天気、湿度、風速風向、降水量、UV指数、24時間予報（横スクロール）、3日間予報。
* 背景テーマ：時間帯（朝・夕・夜）と天気（晴・曇・雨・雪）で自動切替（計10パターン）。
* アニメーション：太陽のパルス、月の光彩、雲の流れ、雨粒、雪の落下をCSSで実装。
* レスポンシブ：モバイル（1カラム）、PC（2カラム）に対応。

## 使い方
HTMLファイルをそのままブラウザーで開けば表示される。
